-> New sesssion: tmux new -s <session-name>
-> Detach session: <ctrl+b>+d
-> List session: tmux ls
-> Reattach session: tmux attach -t <session-name>

Note, you first do <ctrl+b> and then press other keys.

-> <ctrl+b> + c - new window
-> <ctrl+b> + n - next window
-> <ctrl+b> + p - previous window
-> <ctrl+b> + , - rename window
-> <ctrl+b> + w - list windows and select using enter
-> <ctrl+b> + & - kill window or exit or delete

-> <ctrl+b> + " - horizontal split
-> <ctrl+b> + % - vertical split

-> <ctrl+b> + x - close pane

-> <ctrl+b> + o - rotate clockwise pane selection
